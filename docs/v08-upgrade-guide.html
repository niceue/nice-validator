低版本升级到 v0.8+ 指南

Sep 10, 2015

0.8 版本有较大的改变，简化了部分参数和方法，使用方式更加统一。
如果你在使用 nice-validator 过程中没有用到以下特性，则不会受到影响。
如果用到了部分特性，可以参考对应解决办法

不再支持的参数

废弃 groups

解决：使用规则 required(from, class) 替代，参考内置规则 required

废弃 defaultMsg 和 loadingMsg

解决：使用 messages 配置的 fallback 和 loading 替代

不再支持的方法

废弃 mspMsg

解决：直接替换为 showMsg

废弃 setMsg

解决：.attr("data-msg-xxx")

废弃 setRule

解决：.attr("data-rule-xxx")

不再支持的 DOM 属性

废弃 notimely

解决：使用 DOM 属性 data-timely="0"

不再支持的事件

废弃 showtip

解决：使用 .trigger("showmsg", ["tip"]) 替代
