### Build
nice Validator use [gulp](http://gulpjs.com/) to build project.  
you should have installed [nodejs](nodejs.org), cd the `validator` directory, and install dependencies:
```
$ npm install -g gulp
$ npm install
```
Then run gulp will automatic test and build
```
$ gulp
```
